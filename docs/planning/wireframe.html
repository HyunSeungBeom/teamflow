이 문서는 TeamFlow 프로젝트의 전체 개요와 목표를 정리합니다.
프로젝트 목표
Notion + Jira + Slack의 핵심 기능을 하나의 통합 플랫폼으로 제공하여 팀 협업 효율을 극대화합니다.
핵심 가치
하나의 앱에서 문서, 이슈, 채팅을 모두 해결
컨텍스트 스위칭 최소화
실시간 협업과 비동기 소통의 균형
MVP 범위
사용자 인증 (Google OAuth)
프로젝트/워크스페이스 생성
칸반 보드 (드래그 앤 드롭)
문서 에디터 (블록 기반)
실시간 채팅 (채널/DM)
알림 시스템
기술 스택
Frontend: React + TypeScript + Tailwind CSS Backend: Node.js + Express + PostgreSQL Realtime: WebSocket (Socket.io) Infra: AWS (ECS + RDS + S3)
